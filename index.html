Colophon

This is currently held together with a mess of Markdown1 files and HTML edited in BBEdit2 with CSS edited in Sublime Text, managed through the Finder and hosted on GitHub. Someday I may take the time to do this right.

Body text is (ideally) in Geneva (or Verdana), with headings in IBM Plex Sans or Gill Sans.
Theming allows for dark backgrounds with white text in dark mode (the default) or #EEEEEE backgrounds with dark text in light mode (if specifically requested). This theoretically works on any platform that supports light/dark mode (such as Windows 10), but I have only tested it on macOS and iOS.

This website might only be tested in Safari and maybe Firefox.

JavaScript Free (mostly)

This site is mostly JavaScript free. JavaScript is occasionally used to render some features like math or trickier formatting or a map, but it is not required to view the site, and most pages do not require JavaScript. I believe that if JavaScript is needed to view a site, the designer messed up.
