A simple little utility to save Safari tabs from the current window to a file and then open them again later. (Requires an Internet connection: this just saves the URLs, not the page contents.)

To Install

  1. Download the components.
  2. Double-click Save Tabs....workflow to install the service.
  3. Drag Open Safari Tabs to Applications to copy it there.

First-Run

  1. Find Open Safari Tabs in Applications and control-click (right-click) on it and select Open.
  2. Agree to the Gatekeeper prompt to open the application.
  3. Open Safari Tabs will ask for a file. Open the example tabset in the Extras folder to test it.

Granting Permission to Control Safari

  1. To give Open Safari Tabs permission to actually open Safari tabs, go to Apple Menu > System Preferences... and then open Security & Privacy and open the Privacy tab.
  2. Unlock the preference pane by clicking the lock ğŸ”’ at the bottom-left, and then in the list on the left, scroll down and click âš™ï¸Automation.
  3. In the list on the right, find Open Safari Tabs, and make sure Safari underneath is checked â˜‘ï¸ off.

To Use

  1. In Safari, go to Safari > Services > Save Tabs...
  2. Save the tabs somewhere.
  3. Double-click the saved tabs file to open them all again in a new Safari window.

Assigning a Keyboard Shortcut

  1. In System Preferences, go to Keyboard > Shortcuts > âš™ï¸Services, and scroll down to General.
  2. Check â˜‘ï¸ off Save Tabs... and on the right where it says none, double-click to type a keyboard shortcut.
  3. Type the shortcut, and click a different item in the list to save.

System Requirements

This has been tested on macOS Mojave and macOS Big Sur. I cannot help you if it does not work on your system.

This kit is made of two components: a workflow created in Automator that gets all the URLs in the current Safari window and saves them in a list in a text file with a special extension, and an AppleScript packaged into an application using Platypus that reads this text file and opens each URL it encounters in a new tab.
